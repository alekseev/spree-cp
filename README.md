SpreeCp
====================

---


Installation
=======

Add following line to your Gemfile:

    gem 'spree_cp', :git => 'git://github.com/alekseev/spree-cp.git'

Then run this commands:

    rake spree_cp:install:migrations
    rake db:migrate
